# Desenvolver visão geral

Desenvolva um software que se integre com o Acrobat e o Adobe Reader usando JavaScript, plug-ins ou IAC.

## Acrobat SDK

O [SDK do Acrobat](https://opensource.adobe.com/dc-acrobat-sdk-docs/acrobatsdk/){target="_blank"} é um conjunto de ferramentas que ajuda a desenvolver software que interage com a tecnologia Acrobat.
