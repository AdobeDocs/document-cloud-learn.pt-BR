# Quando usar a opção Criar uma cópia

Já enviou um contrato — mas ele expirou, foi cancelado ou você precisa enviá-lo novamente? Criar uma cópia permite duplicar qualquer contrato existente, ajustar o que precisa ser alterado e reenviá-lo em segundos. Não recriar do zero. Criar uma cópia está disponível quando:

* Você é o remetente original do contrato
* O contrato foi enviado da experiência moderna de solicitação de assinatura
* O contrato não inclui uma função de destinatário testemunha
* O contrato não foi excluído devido à retenção de documentos ou exclusão do GDPR
* O contrato não requer acesso a um modelo de biblioteca ao qual o remetente não tem mais acesso

>[!NOTE]
>
>Criar uma cópia está disponível para os planos corporativos.
